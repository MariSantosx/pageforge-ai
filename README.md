# PageForge AI 🚀

Gerador de landing pages profissionais com Inteligência Artificial.

## O que faz?
Digite o tema do seu negócio e receba um site completo em segundos, gerado por IA.

## Tecnologias usadas
- HTML
- CSS
- JavaScript
- API do Groq (LLaMA 3.3)

## Como usar?
1. Digite o tema do seu negócio no campo de texto
2. Clique em "Gerar"
3. Aguarde a IA criar seu site completo
