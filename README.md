# PageForge AI 🚀

Gerador de landing pages profissionais com Inteligência Artificial.

## 💡 Sobre o Projeto
O PageForge AI é uma aplicação web que utiliza Inteligência Artificial para gerar landing pages completas e profissionais em segundos. Basta descrever o seu negócio e a IA cria um site inteiro com design moderno, informações relevantes, seções organizadas e muito mais.

## ✨ Funcionalidades
- Geração automática de sites completos com HTML, CSS e JavaScript
- Design moderno e responsivo
- Seções completas: navbar, hero, sobre, produtos, depoimentos, FAQ, contato e footer
- Imagens automáticas relacionadas ao tema
- Ícones profissionais com Font Awesome
- Cores e gradientes modernos
- Código pronto para usar

## 🛠️ Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript puro
- API do Groq (modelo LLaMA 3.3 70B)

## 🚀 Como Usar
1. Acesse o projeto no navegador
2. Digite o tema do seu negócio no campo de texto (ex: Cafeteria, Loja de Roupas, Academia)
3. Clique no botão **⚡ Gerar**
4. Aguarde a IA processar e gerar seu site completo
5. Visualize o resultado em tempo real
